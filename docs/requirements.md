# 要件定義

## 概要

同棲しているカップルや複数人世帯向けの家計簿Webアプリ。支出、支払者、カテゴリ、割り勘率を記録し、誰がいくら多く払っているか、誰が誰へいくら渡すとよいかを表示する。

## 対象環境

- スマホWebブラウザ: Safari、Chrome
- PC Webブラウザ: Chrome
- 自宅サーバー上のDocker運用
- 非公開利用を前提とし、初期版ではログインなし

## 初期MVP

- メンバー管理
  - 名前の追加、編集、無効化
- カテゴリ管理
  - カテゴリ追加、編集
  - カテゴリ並び替え
  - デフォルトカテゴリ: 家賃、光熱費、通信費、食費、その他
  - メンバーごとのデフォルト割り勘率を保存
- 支出管理
  - 支出内容、カテゴリ、割り勘率、金額、支払者を登録
  - 支出ごとの割り勘率はカテゴリ選択時に反映し、個別編集できる
  - 未計上支出を既定表示し、チェックした支出を清算計上できる
- 清算管理
  - 清算計上済み支出とお金受け渡し履歴を合算
  - メンバー別の支払済み、負担額、過不足を表示
- 自動支出
  - 毎日、毎週、毎月のルールを登録
  - アプリ内workerが対象日に支出を自動生成
- 外部連携の土台
  - 支出作成と清算サマリー取得のロジックをAPI/Webhookから再利用できるよう分離

## 初期版で含めないこと

- LINE Bot本体
- 外部公開前提の認証、署名検証、レート制限
- 複数世帯や複数家計簿の切り替え
